## Uso da Collection

### MCP M365 Agent Expert

recommended

Este chat mode fornece orientacao especialista para construir agentes declarativos baseados em MCP para Microsoft 365 Copilot.

Este chat mode e ideal para:
- Criar novos agentes declarativos com integracao MCP
- Projetar Adaptive Cards para respostas visuais
- Configurar autenticacao OAuth 2.0 ou SSO
- Configurar response semantics e data extraction
- Solucionar problemas de deployment e governanca
- Aprender boas praticas de MCP para M365 Copilot

Para obter os melhores resultados, considere:
- Usar o arquivo de instruction para definir o contexto para todas as interacoes com Copilot
- Usar prompts para gerar a estrutura inicial do agent e configuracoes
- Alternar para o chat mode especialista para ajuda detalhada de implementacao
- Fornecer detalhes especificos sobre seu MCP server, tools e cenario de negocio
