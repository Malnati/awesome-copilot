## About This Collection

Esta collection de 7 agents e baseada em aprendizados dos experimentos [The AI-Native Engineering Flow](https://medium.com/data-science-at-microsoft/the-ai-native-engineering-flow-5de5ffd7d877) da Microsoft, projetada para ampliar equipes de engenharia de software ao longo de todo o ciclo de vida de desenvolvimento.

**Principios de Design Principais:**
- **Standalone**: Cada agent funciona de forma independente sem dependencias cruzadas
- **Enterprise-ready**: Incorpora frameworks OWASP, Zero Trust, WCAG e Well-Architected
- **Cobertura do ciclo de vida**: De pesquisa UX -> Arquitetura -> Desenvolvimento -> Seguranca -> DevOps

**Agents nesta collection:**
- **SE: UX Designer** - Analise Jobs-to-be-Done e mapeamento de jornada do usuario
- **SE: Tech Writer** - Documentacao tecnica, blogs, ADRs e guias de usuario
- **SE: DevOps/CI** - Debugging de CI/CD e troubleshooting de deployment
- **SE: Product Manager** - GitHub issues com contexto de negocio e criterios de aceitacao
- **SE: Responsible AI** - Testes de vies, acessibilidade (WCAG) e desenvolvimento etico
- **SE: Architect** - Reviews de arquitetura com frameworks Well-Architected
- **SE: Security** - OWASP Top 10, seguranca LLM/ML e Zero Trust

Voce pode usar agents individuais conforme necessario ou adotar a collection completa para ampliacao abrangente da equipe.
