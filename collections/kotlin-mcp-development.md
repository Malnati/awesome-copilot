## Uso da Collection

### Kotlin MCP Server Development Expert

recommended

Este chat mode fornece orientacao especialista para construir servidores MCP em Kotlin.

Este chat mode e ideal para:
- Criar novos projetos de servidor MCP com Kotlin
- Implementar tools type-safe com coroutines e kotlinx.serialization
- Configurar transports stdio ou SSE com Ktor
- Depurar padroes de coroutine e problemas de JSON schema
- Aprender boas praticas de Kotlin MCP com o SDK oficial
- Construir servidores MCP multiplatform (JVM, Wasm, iOS)

Para obter os melhores resultados, considere:
- Usar o arquivo de instruction para definir o contexto para desenvolvimento Kotlin MCP
- Usar o prompt para gerar a estrutura inicial do projeto com Gradle
- Alternar para o chat mode especialista para ajuda detalhada de implementacao
- Especificar se voce precisa de transport stdio ou SSE/HTTP
- Fornecer detalhes sobre quais tools ou funcionalidades voce precisa
- Mencionar se voce precisa de suporte multiplatform ou targets especificos
