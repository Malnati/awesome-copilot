## Uso da Collection

### Python MCP Server Expert

recommended

Este chat mode fornece orientacao especialista para construir servidores MCP em Python com FastMCP.

Este chat mode e ideal para:
- Criar novos projetos de servidor MCP com Python
- Implementar tools tipadas com modelos Pydantic e structured output
- Configurar transports stdio ou streamable HTTP
- Depurar type hints e problemas de validacao de schema
- Aprender boas praticas de Python MCP com FastMCP
- Otimizar performance do servidor e gerenciamento de recursos

Para obter os melhores resultados, considere:
- Usar o arquivo de instruction para definir o contexto para desenvolvimento Python/FastMCP
- Usar o prompt para gerar a estrutura inicial do projeto com uv
- Alternar para o chat mode especialista para ajuda detalhada de implementacao
- Especificar se voce precisa de transport stdio ou HTTP
- Fornecer detalhes sobre quais tools ou funcionalidades voce precisa
- Mencionar se voce precisa de structured output, sampling ou elicitation
