# GitHub Copilot Cookbook

Uma colecao de receitas e exemplos praticos para trabalhar com ferramentas e recursos do GitHub Copilot. Cada receita oferece trechos de codigo focados e prontos para copiar e colar, com exemplos do mundo real para ajudar voce a concluir tarefas comuns.

## O que ha no Cookbook

O cookbook e organizado por ferramenta ou produto, com receitas agrupadas por linguagem quando aplicavel:

### GitHub Copilot SDK

Receitas prontas para uso para construir com o GitHub Copilot SDK em varias linguagens.

- **[Copilot SDK Cookbook](copilot-sdk/)** - Receitas para .NET, Go, Node.js e Python
  - Tratamento de erros, gerenciamento de sessoes, operacoes de arquivo e mais
  - Exemplos executaveis para cada linguagem
  - Melhores praticas e guias completos de implementacao

## Como Comecar

1. Navegue ate a pasta da ferramenta ou produto que combina com o que voce quer construir
2. Encontre a receita que resolve o seu caso de uso
3. Copie o trecho de codigo ou verifique a subpasta `recipe/` para exemplos completos e executaveis
4. Consulte a documentacao especifica da linguagem para configuracao e instrucao de execucao

## Expansoes Planejadas

O cookbook foi projetado para crescer junto ao ecossistema do GitHub Copilot. Adicoes futuras podem incluir receitas para:

- Ferramentas e integracoes adicionais do Copilot
- Padroes e fluxos de trabalho avancados
- Integracao com servicos externos e APIs
- Otimizacoes especificas de linguagem e melhores praticas
